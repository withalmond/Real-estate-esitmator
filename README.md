# Real Estate Estimator

Property walkthrough and estimation tools.

## Getting started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) and choose **Start Property Walkthrough**, or go directly to [http://localhost:3000/walkthrough/new](http://localhost:3000/walkthrough/new).

## Property Walkthrough (`/walkthrough/new`)

1. Enter the property address at the top.
2. Use **Add room** to build an interactive list of spaces.
3. For each room: select type, enter dimensions / square footage, add dictated notes, and capture photos with the device camera. Thumbnails appear under each room as photos are taken.
